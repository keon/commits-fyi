# Repository Stars Forks
How it works. A GitHub Actions cron job refreshes this data daily. For each city, we search GitHub for users/orgs whose self-reported profile location matches the city, then sum stargazers across their public non-fork repos. Location is free-text — e.g. New York won't match users who wrote NYC or Brooklyn. View source
